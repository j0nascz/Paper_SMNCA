\section{Introduction}
\subsection{Motivation}
\subsubsection{Why statistical network analysis is important}
We live in a highly interconnected world where 
many physical, social, technological and 
biological systems consist of agents
or entities interacting with each other.
Examples include a virus being transmitted 
over social contact networks, global trade between 
countries, and the human brain. 
Any such system can be represented as a network 
by denoting the entities as nodes and the 
interactions between them as edges. 
This makes networks an important type of data 
spanning a remarkable variety of complex systems. 
It is therefore very important to have 
mathematically rigorous and practically 
useful methods for statistical analysis of networks.


\subsubsection{Biological Networks}
Biological systems are often represented as networks. Essentially,
every biological entity has interactions with other biological entities,
from the molecular to the ecosystem level, providing us with the opportunity
to model biological systems using many different types of networks such as 
ecological, neurological, metabolic, and protein–protein interaction networks.

A protein-protein interaction network (PPI network), 
also known as the interactome,
is a type of biological network where a node represents 
a protein and an edge represents the physical 
interaction between each pair of proteins.

\subsection{Definition of a Network}
\subsubsection{Graph as  a mathematical object}
We call a collection of connected objects a network. In the field of mathematics,
a network can also be called a graph. The mathematical theory behind networks is known as graph theory.
A graph is a mathematical object that consists of a set of objects (called vertices or nodes) that are
connected by links (called edges).

\subsubsection{Adjacency list}

An adjacency list is a graph representation method that uses an array of lists to store connections between vertices. 
Each array index represents a vertex, and its corresponding list contains all 
neighboring nodes, making it highly efficient for sparse graphs($G= (V,E)$, where the number
of edges is much less than the maximum possible number of edges).
The main operation performed by the adjacency list data structure is to report a list of neighbors
of a given vertex.

\subsubsection{Adjacency matrix}
The main alternative to the adjacency list is the adjacency matrix,
a matrix whose rows and columns are indexed by vertices and the cell $(i,j)$ 
contains 1 if there is an edge between vertices $i$ and $j$, or 0 if no such edge exists.
The diagonal elements of the matrix are all 0, since
edges from a vertex to itself are not allowed in simple graphs.

\subsubsection{Directed vs.\ undirected networks}
Undirected and directed graphs are fundamental concepts in graph theory.
An undirected graph is a graph where the edges do not have a specific direction and
it is bidirectional in nature meaning it does not have a parent-child relation concept.
In contrast, a directed graph is a graph where the edges have a specific direction,
indicating a one-way relationship between the vertices.

An important difference between directed and undirected graphs is that in a undirected graph,
relationships are symmetric, meaning if vertex A is connected to vertex B, 
then vertex B is also connected to vertex A. In a directed graph, relationships are asymmetric, meaning 
if vertex A is connected to vertex B, it does not necessarily imply that vertex B is connected to vertex A.

\subsubsection{Weighted undirected network}
Because in this paper we will be working with weighted undirected networks,
we will now define what a weighted undirected network is.
A weighted undirected network is a graph where each edge has a weight associated with it, 
representing the strength or importance of the connection between two vertices. 
In such networks, the adjacency matrix is not binary but contains numerical weights for each edge.
Mathematically, a weighted undirected network can be represented as $G = (V, E, W)$,
where $V$ is the set of vertices, $E$ is the set of edges, and $W$ is a function that assigns a weight 
to each edge in $E$.

\subsection{Network Analysis}
\subsubsection{Descriptive statistics}
\textcite{DavidHevey2018} states that Network analysis provides the capacity to 
estimate complex patterns of relationships, and the network structure can be analyzed to reveal core 
features of the network.
 A variety of basic descriptive metrics can be used to characterize network topology, including
number of nodes and edges, network density, degree distribution and centrality measures.
These measures provide an initial understanding of connectivity patterns and structural organization.

\subsubsection{Community Detection and Community Comparison}
In network science, a community is a group of nodes that are more densely connected to each other than
to the rest of the network. Detecting communities can help us understand the functional 
organization of a system. For example, in a PPI network, groups of proteins that interact often perform
the same function in a cell.
Community detection methods such as Edge-Betweenness, Louvain and Leiden algorithms 
can be used to identify these community structures.
For this paper, 
attention is given to the Adjusted Rand Index (ARI) as a similarity measure for community comparison, 
which quantifies the agreement between different community structures. 
This concept will be explained in more detail in the following sections.
